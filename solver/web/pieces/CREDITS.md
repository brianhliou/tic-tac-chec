# Piece credits

The explorer uses the **Cburnett** chess piece set by Colin M.L. Burnett,
distributed here under GPL-3.0-or-later. These eight SVGs are the pawn, knight,
bishop, and rook artwork extracted from the assets embedded by
[`chessground`](https://github.com/lichess-org/chessground) 9.2.1 and used as
Mistboard's default chess piece set.

The original solver and explorer source is MIT-licensed. The artwork remains
under its separate GPL terms; the complete license text is included in
[`LICENSE`](LICENSE).
